Qué formato deben tener los datos de manzanas

El formato soportado para los datos de manzanas es el shapefile (.shp), los features deben contener los atributos que se muestra en la tabla 1.

Tabla 1: Atributos de los features para representar manzanas
Nombre Tipo                     Descripción
``the_geom'' Polygon Figura que representa la manzana. Se define mediante las coordenadas de sus vértices, en metros. Es la default geometry.
