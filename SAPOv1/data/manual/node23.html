Qué formato deben tener los datos de edificaciones

El formato soportado para los datos de edificaciones es el shapefile (.shp), los features deben contener los atributos que se muestra en la tabla 2.

Tabla 2: Atributos de los features para representar edificios
Nombre Tipo                     Descripción
``the_geom'' Polygon Figura en dos dimensiones que representa la base del edificio. Se define mediante las coordenadas de sus vértices, en metros. Es la default geometry.
``altura'' double Valor numérico que representa la altura del edificio en metros.
