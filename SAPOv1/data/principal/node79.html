Archivos de entrada de datos de edificaciones y manzanas

Para manejo de datos vectoriales, uno de los formatos más populares es el shapefile (.shp). Es el único formato de este tipo soportado en la distribución utilizada de GeoTools tanto para escritura como para lectura, por lo cual se decidió utilizarlo para los datos de edificaciones y de manzanas. La posibilidad de escribir archivos es de interés a fin de dar herramientas al usuario de crear sus propias bases de datos mediante una interfaz gráfica (el Generador de Edificios).
